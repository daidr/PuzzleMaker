一、CQPM的界面

emmm...希望CQPM的界面没有给你很大的压力（至少我自己看来还是可以哒）

编辑器中间面积最大的是拼图编辑区，所有的拼图操作都将在此区域进行。此区域的右下角是界面的调整按钮。

在编辑器的左侧，是 拼图工具栏，你可以将需要的拼图从中拖动到编辑器的拼图编辑区。将鼠标长时间停留在某个拼图上，会出现该拼图的简单介绍。

编辑器右上角是 菜单组，常用的操作按钮都集成在此区域内，集合了一些基本操作。每个按钮的作用将在「基本操作」章节讲到。

二、CQPM的基本操作

CQPM编辑器提供了十分便捷的代码编辑方式。目前，你能够通过本编辑器，来自定义 酷Q应用:“Puzzle Plugin” 的功能。多插件机制在努力开发中，敬请期待。

①拼图工具栏：
拼图工具栏的唯一用途就是让你将需要的拼图加入到编辑区中。拼图工具栏有多个分类(如“逻辑” “循环” “CQ事件”)，单击每个分类，将会出现该分类下的所有拼图。使用鼠标将需要的拼图拖动到拼图编辑区即可。

②拼图编辑区：
所有的拼图操作都在此区域进行。首先，每个拼图都有自己的槽口，你需要做的就是理清逻辑关系，将需要的拼图通过槽口连接到一起，从而形成一段完整的代码。
在拼图(或空白处)右键来调出右键菜单，你可以通过菜单来对某拼图(或整个编辑区)进行操作，比如(“注释块” “禁用块” “删除块”)
该区域右下角是布局调整按钮以及垃圾箱。你能够通过布局调整按钮放大、缩小以及居中编辑区，将拼图(或拼图组)拖动到垃圾箱上方可快速删除拼图。

③拼图工具栏：
拼图工具栏的唯一用途就是让你将需要的拼图加入到编辑区中。拼图工具栏有多个分类(如“逻辑” “循环” “CQ事件”)，单击每个分类，将会出现该分类下的所有拼图。使用鼠标将需要的拼图拖动到拼图编辑区即可。鼠标停留在某块拼图上来查看拼图作用。

④菜单组：
菜单组里有一些基本的操作按钮，下面将一一介绍。
live_help : 点击来调出/隐藏帮助文档
cloud_upload : 点击后会生成一段唯一的绑定码，将绑定码输入酷Q插件即可完成插件功能定义
publish : 将一个拼图布局文件追加到目前布局的后面（不清除当前已有拼图）
file_upload : 将一个拼图布局文件导入到拼图编辑器（将覆盖编辑区的拼图）
save : 将目前编辑区的拼图布局导出为文件，方便分享代码以及备份代码。
sd_card : 将目前编辑区的拼图布局储存到浏览器缓存中（缓存不会因为浏览器关闭而消失）。暂时离开电脑或关闭浏览器时，要记得点击该按钮，避免当浏览器崩溃时功归一篑。但是请注意，本功能只能作为暂时储存，如果想要备份/长时间保存，请使用 导出到本地文件 的功能，浏览器缓存可能会因为升级浏览器/清理垃圾而丢失。
